# 命名管道传输（Windows）

实现 `NamedPipeTransport`，按行收发 JSON 文本。

## 使用
```
var transport = new NamedPipeTransport("axolotl-mcp");
await transport.StartAsync();
await transport.SendAsync("{\"ping\":true}");
await foreach (var msg in transport.ReadAsync())
{
    Console.WriteLine(msg);
}
```

## 注意事项
- 仅 Windows 可用；服务端需已创建同名管道。
- 文本编码 UTF-8，无 BOM；按行分隔。
- 异常或关闭时读取将结束。
